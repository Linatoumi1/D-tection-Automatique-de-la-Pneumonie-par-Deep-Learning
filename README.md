# D-tection-Automatique-de-la-Pneumonie-par-Deep-Learning
une étude comparative approfondie de cinq architectures de Deep Learning pour la classification binaire d'images de radiographies thoraciques (Normal vs Pneumonie). Les modèles étudiés incluent un CNN personnalisé avec Dropout, MobileNet, ResNet50, VGG16, et un modèle hybride innovant ResNet-ViT combinant convolutions et transformateurs.
